Three-dimension-reconstruction

The project aims on creating an application that can generate 3D models of landmarks from 2D images images that are available in public datasets.

Project Description

The aim of our project is to construct 3d models of a landmark from publicly available 2d images of the landmark.

Abstract

Introduction

Three Dimensional reconstruction can be considered as a more expressive mechanism when look at a large collection of unstructured photographs. It can be applied to reconstruct landmarks like Notre-Dame Cathedral or Statue of Liberty from a large dataset of images that is available publicly. The Three Dimensional Representation give a more accurate perspective of the features of the landmarks by showing its different facets in a single representation.

Input Data

The project uses large number of publicly available images that can be obtained from either flickr or Google images. Dataset used comprises of unstructured images of a landmark from different views.

Algorithm Used

The main algorithm used in the project is

  • Feature Matching: SIFT and approximate nearest neighbour
  • Determining Camera Parameter: Structure From Motion
